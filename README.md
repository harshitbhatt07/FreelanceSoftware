# FrelanceSofteare - Webtonic Style Portfolio

Modern React + Tailwind CSS portfolio website inspired by premium agency websites. Content is for FrelanceSofteare.

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Contact Form
Copy `.env.example` to `.env` and fill EmailJS values:
```env
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```
The contact form sends fields to `himanshubhatt050505@gmail.com` through your EmailJS template.
